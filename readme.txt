Technical Notes:

	Pre-requisites : sbt should be involved.

	Run the project using below commands:
		1. sbt update
		2. sbt compile
		3. sbt run

	To run tests use the coomand
		sbt test

	To check code coverage using Jacoco use below coomand
		sbt jacoco

	To generate java doc for source file
		sbt doc
	
	To generate java doc for test classes
		sbt test:doc

The application will be hosted at http://localhost:9000/
